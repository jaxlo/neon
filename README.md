# neon
A raytraced social game built with C++ and Vulkan

Dependencies
- Vulkan
- SDL3
- FTXUI


## Runing neon
``sudo dnf install SDL3 ftxui``

``mkdir build``
``cd build``

### Generate build files
``cmake ..``

### Compile the project
``make``

Or on Windows with Visual Studio: 

``cmake --build .``
